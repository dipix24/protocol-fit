PROTOCOL FIT PWA — ISTRUZIONI RAPIDE
====================================

Questa cartella contiene una Progressive Web App pronta per GitHub Pages.
Carica I FILE CONTENUTI in questa cartella nella radice del repository.
Non caricare soltanto il file ZIP.

FILE PRINCIPALI
- index.html
- styles.css
- app.js
- plan.json
- manifest.webmanifest
- service-worker.js
- icons/

PUBBLICAZIONE GRATUITA CON GITHUB PAGES
1. Crea un account gratuito su github.com.
2. Crea un nuovo repository pubblico, per esempio “protocol-fit”.
3. Nel repository scegli Add file > Upload files.
4. Trascina tutti i file e la cartella icons presenti qui.
5. Conferma con Commit changes.
6. Apri Settings > Pages.
7. In Build and deployment seleziona Source: Deploy from a branch.
8. Seleziona Branch: main e cartella /(root), quindi Save.
9. Attendi alcuni minuti. GitHub mostrerà il link pubblico della PWA.

INSTALLAZIONE SU IPHONE
1. Apri il link con Safari.
2. Tocca il pulsante Condividi.
3. Tocca “Aggiungi alla schermata Home”.
4. Lascia il nome “Protocol Fit” e tocca Aggiungi.
5. Aprila dalla nuova icona.
6. Aprila una prima volta con internet; in seguito può funzionare offline.

PRIVACY
I pesi, le ripetizioni, il workout in corso e lo storico vengono salvati
nel browser del dispositivo. Non vengono caricati nel repository GitHub.
Usa Impostazioni > Esporta per creare periodicamente un backup JSON.

AGGIORNAMENTI
Per aggiornare l’app, sostituisci i file nel repository. Dopo la pubblicazione,
chiudi completamente Protocol Fit e riaprila. In caso di cache persistente,
rimuovi e aggiungi nuovamente l’icona alla schermata Home.
